# MarkDown Tutorial

### Introduction

markdown is  a lightweight markup language that uses simple, plain-text formating to create structured documents that can be easily converted  to HTML

Created by John Gruber in 2004, its design goal  is to be human-readable and writeble, making it esay to add formatting like headings, lists, and bold text using basic symbols.

Markdown is widely used for creating technical documntation , web content, readme files, and content for platform like Github, Azure Devops, and online forms.

### How to create And preview Markdown file?

- Create a new file and save it with a.md Extention
- Open File in VS Code
- Press Ctrl+K V (Window/linux),press cmd+K V (macos)
